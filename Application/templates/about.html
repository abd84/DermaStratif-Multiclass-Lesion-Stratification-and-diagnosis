DermaStratif: Multiclass Lesion Stratification and Diagnosis


Base Model: EfficientNet

The core architecture of the skin cancer classification model is built upon EfficientNet-B0, a lightweight yet highly effective convolutional neural network that optimizes both accuracy and computational efficiency.

  • Architecture: EfficientNet-B0
  • Pretrained Weights: Trained on the ImageNet dataset
  • Purpose: To act as the backbone for feature extraction
  • Base Model Validation Accuracy: 74%

Fine-Tuning Techniques

The model was fine-tuned using advanced techniques to adapt the base architecture for the specific task of classifying skin cancer types:

1. LoRA (Low-Rank Adaptation)

LoRA introduces low-rank learnable matrices into the pre-trained model layers to enable efficient fine-tuning. This technique allows the model to learn task-specific features with minimal computational overhead.

  • LoRA layers were applied to the fully connected layers in the classifier.
  • Enabled efficient parameter tuning without modifying the pre-trained backbone.

Validation Accuracy: 80%

2. Adapter Layers

Adapter layers were added to the base model to learn additional task-specific parameters while freezing most of the original model weights. This ensures the base model remains largely intact while improving classification performance.

  • Adapter layers were introduced in intermediate feature layers of the model.
  • Improved the model's ability to generalize across different types of skin lesions.

Validation Accuracy: 76%

Dataset

The model was trained and validated on a dataset containing high-resolution dermoscopic images of skin lesions. The dataset included images from the following classes:

  • Melanoma
  • Basal Cell Carcinoma
  • Squamous Cell Carcinoma
  • Actinic Keratosis
  • Benign Keratosis
  • Dermatofibroma
  • Vascular Lesion
  • Melanocytic Nevus

The dataset was preprocessed with augmentation techniques such as resizing, rotation, and normalization to improve model robustness.

Summary of Model Performance

Base Model Validation Accuracy: 74%
Adapter Fine-Tuning Validation Accuracy: 76%
LoRA Fine-Tuning Validation Accuracy: 80%
